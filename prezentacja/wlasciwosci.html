@media queries

@media queries to funkcja CSS 3 umożliwiająca renderowanie treści w celu dostosowania ich do różnych warunków, takich jak rozdzielczość ekranu. @media queries się standardem zalecanym przez W3C w czerwcu 2012 roku i jest podstawą technologii responsywnego projektowania stron internetowych.

Zmienne CSS

Zmienne CSS pomagają uniknąć powtórzeń, ułatwiają ponowne wykorzystanie stylów oraz wyeliminowanie niespójności. Wszystko, co trzeba zrobić, to zadeklarować je raz i wywołać w miejscu, w którym chcemy ich użyć. Prawidłowe określenie zmiennych to przypisanie im wartości w ten sposób:

:root{
--color: yellow;
--width: 80vw;
}

Prawidłowe zastosowanie tych zmiennych jest następujące:

article{
background-color: var(--color);
max-width: var(--width);
}

Jednostki vw i vh.

W języku CSS dzięki tym jednostkom możemy określać wielkość elementów w pełni responsywny sposób. Jednostki vh i vw są jednostkami które są wyznaczane na podstawie wysokości i szerokości okna przeglądarki. Np: jeżeli okno będzie mieć 1000px szerokości i 700px wysokości to 1vh będzie równy 7px, a 1vw będzie równy 10px. Używanie vh jest jednym z niewielu możliwości responsywnego określania wysokości elementu. Zastosowanie:

body{
width: 100vw;
height: 100vh;
margin: 0 !important;
padding: 0 !important;
}

section{
height: 30%;
width: 80%;
}

Position: sticky;

Sticky jest jedną z wartości, które może przyjmować position. Jest używane wtedy gdy chcemy by nasz element przykleił się do określonej części okna. Najlepszym przykładem właściwości position: sticky; jest navigacja na tej stronie. Przykładowa składnia CSS, pozwalająca na stworzenie takiego elementu:

nav{
position: sticky;
position: -webkit-sticky;
top: 0;
width: 100%;
padding: 10px 0;
background-color: #ff3347;
text-align: center;
border-top: 1px solid #fa55dd;
border-bottom: 1px solid #fa55dd;
min-height: 40px;
z-index: 100;
}

Tekst z gradientem

Tekst na stronie z gradientem robimy na stronie aby w sposób przyzwoicie wyglądający wyróżnić Pewne słowa w tekście. Taką właściwość można uzyskać wpisując następujące linijki w CSS:

span{
display: block;
margin-top: 16px;
text-align: center;
font-size: 32px;
font-weight: bolder;
background: linear-gradient(90deg, green, white);
color: transparent;
-webkit-background-clip: text;
background-clip: text;
}
